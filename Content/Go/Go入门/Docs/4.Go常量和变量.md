
# 常量

```go
// ========= 常量 ===========
// 1. 定义一个常量
const Pi = 3.14159 

// 2. 定义枚举常量
// iota被当做枚举值，第一个为0。
// 在每次遇到const关键字，iota都会重置为0。
const (
	a = iota // 0
	b        // 1
	c				 // 2
)

// 3. 输出
fmt.Println(a, b, c, Pi)
// 输出：0 1 2 3.14159

```
